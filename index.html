Margin (Outer space)

Margin creates space outside the element's border, pushing other elements away.

Use margin-top to add space above an element.

Margin Top (20px)

Use margin-right to add space to the right of an element.

Margin Right (20px)

Use margin-bottom to add space below an element.

Margin Bottom (20px)

Use margin-left to add space to the left of an element.

Margin Left (20px)

Use the shorthand margin to set all sides at once (top, right, bottom, left).

Margin All (20px)

Shorthand margin values can be set individually like margin: 10px 15px 20px 25px; (top, right, bottom, left).

Margin Shorthand (10px 15px 20px 25px)

Padding (Inner space)

Padding creates space inside the element, between the content and the border.

Use padding-top to add space above the content inside the element.

Padding Top (20px)

Use padding-right to add space to the right inside the element.

Padding Right (20px)

Use padding-bottom to add space below the content inside the element.

Padding Bottom (20px)

Use padding-left to add space to the left inside the element.

Padding Left (20px)

The shorthand padding sets padding on all sides at once.

Padding All (20px)

Padding shorthand can specify different values for each side like padding: 5px 10px 15px 20px; (top, right, bottom, left).

Padding Shorthand (5px 10px 15px 20px)
